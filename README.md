# CREDIT-CARD-FINANCIAL-DASHBOARD
Credit Card Customer Report & Credit Card Transaction Report
A dynamic, interactive data visualization tool built to explore credit card data-focusing on revenue, income, interest and customer satisfaction score.

DESCRIPTION:
The Credit Card Financial Dashboard is a visually engaging, Power BI business intelligence project designed to analyze customer behavior, credit card usage, transaction activity, and revenue performance. The dashboard provides actionable insights into customer demographics, transaction trends, utilization behavior, and overall financial performance.

OBJECTIVE:
The main objectives of this dashboard are:
-Provide a centralized view of credit card business performance.
-Analyze customer demographics and spending behavior.
-Track revenue generation across customer segments.
-Monitor credit card utilization and transaction volume.
-Enable trend analysis using weekly and quarterly metrics.

TECH STACK:
The dashboard was built using the following tools and technologies:-
Power BI Desktop: Main Data visualization used for dashboard creation
Power Query:	Data transformation and cleaning layer for reshaping and preparing the data.
DAX (Data Analysis Expressions): Used for	KPI calculated measures, dynamic visuals, and conditional logic.
Data Modeling:	Relationship establishment among tables to enable cross-filtering and aggregation.
File Format: Developed using Microsoft Power BI and packaged as a .pbit Power BI Template file.

FEATURES:
-Comparative week-over-week performance analysis
-Revenue analysis by card category and expense type
-Customer satisfaction and delinquency tracking
-Time intelligence reporting using weekly and quarterly metrics
-Cross-filtering between visuals for deeper insights

HIGHLIGHTS:
-Provided actionable insights into customer spending behavior
-Improved decision-making with KPI-driven dashboards
-Designed scalable dashboard architecture for future enhancements
-Suitable for banking, fintech, and financial services analytics use cases

BUSINESS IMPACT & INSIGHTS:
- Attentive:- Helps identify underperforming customer segments and business areas requiring attention
- Planning finance:- Supports forecasting and future financial planning using historical transaction trends
- Retain customers:- Improves customer retention strategies by analyzing satisfaction scores and usage patterns
- Monitoring business: Helps management monitor overall business performance through real-time KPIs and revenue insights
